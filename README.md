# Cal.com API v2 Monorepo

This monorepo contains the following packages:

- **[SDK](packages/sdk)**: TypeScript SDK for Cal.com v2 API.
- **[MCP Server](packages/mcp-server)**: Model Context Protocol server for Cal.com (In Development).

## Development

This project uses [Bun](https://bun.sh) workspaces.

### Install dependencies

```bash
bun install
```

### Build all packages

```bash
bun run build
```

### Test all packages

```bash
bun run test
```
